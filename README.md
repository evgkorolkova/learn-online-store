# learn-online-store

1) Пункт
	Создать своё виртуальное окружение с имененм venv
	python3 -m venv venv
2) Пункт
	Активировав окружение
	venv\Scripts\activate
3) Пункт
	Апргейдить pip
	python3 -m pip install --upgrade pip
4) Пункт
	Установка Django и пакетов внутри проекта
	pip install -r requirements.txt
